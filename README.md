# Equipment Failure Prediction Platform

An end-to-end industrial equipment failure prediction platform with:

- predictive maintenance ML models
- multi-agent backend orchestration
- explainable AI
- LLM-powered chatbot
- FastAPI backend
- Lovable/Bolt-based frontend

## Phase 1
Backend project setup:
- FastAPI skeleton
- config management
- logging
- health endpoint
- project structure

## Run locally

1. Create virtual environment
2. Install dependencies with `pip install -r requirements.txt`
3. Run the app with `python run.py`
